---
title: 'Get Reports'
api: 'GET /reports'
---

Retrieves a list of reports with optional filtering by status, ID, and pagination support.

## Query Parameters

<ParamField query="status" type="string" optional>
  Filter reports by status. Accepts one of the following values:
  - `1` - Pendiente (Pending)
  - `2` - Aprobada (Approved)
  - `3` - Rechazada (Rejected)
</ParamField>

<ParamField query="id" type="string" optional>
  Filter reports by specific report ID. Must be a positive integer.
</ParamField>

<ParamField query="page" type="number" optional>
  Page number for pagination. Must be a positive integer. Returns paginated results when specified.
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the report
</ResponseField>

<ResponseField name="title" type="string">
  Title of the report
</ResponseField>

<ResponseField name="image" type="string">
  Path to the report's image file
</ResponseField>

<ResponseField name="description" type="string">
  Detailed description of the reported issue
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp when the report was created
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp when the report was last updated
</ResponseField>

<ResponseField name="user_name" type="string">
  Name of the user who created the report
</ResponseField>

<ResponseField name="created_by" type="number">
  User ID of the report creator
</ResponseField>

<ResponseField name="user_image" type="string">
  Path to the creator's profile image
</ResponseField>

<ResponseField name="report_url" type="string">
  URL of the reported website or resource
</ResponseField>

<ResponseField name="categories" type="array">
  Array of category IDs associated with this report
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.falconalert.com/reports?status=1&page=1'
```

```javascript JavaScript
const response = await fetch('https://api.falconalert.com/reports?status=1&page=1', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const reports = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.falconalert.com/reports',
    params={'status': '1', 'page': 1}
)

reports = response.json()
```

</RequestExample>
