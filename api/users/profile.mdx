---
title: "Get User Profile"
api: "GET /users"
description: "Retrieve the authenticated user's profile information"
---

## Authentication

This endpoint requires a valid JWT Bearer token. The profile returned corresponds to the authenticated user.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  Format: `Bearer <your_access_token>`
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique user identifier
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="name" type="string">
  User's full name
</ResponseField>

<ResponseField name="password" type="string">
  Hashed password (for security)
</ResponseField>

<ResponseField name="salt" type="string">
  Password salt used for hashing
</ResponseField>

<ResponseField name="image_path" type="string">
  Path to user's profile image
</ResponseField>

<ResponseField name="role_id" type="number">
  User's role identifier
  
  - `1` - Regular user
  - `2` - Admin user
</ResponseField>

<ResponseField name="created_at" type="string">
  Account creation timestamp (ISO 8601 format)
</ResponseField>

<ResponseField name="updated_at" type="string">
  Last profile update timestamp (ISO 8601 format)
</ResponseField>

<ResponseField name="username" type="string">
  User's unique username
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.falconalert.com/users \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```javascript JavaScript
const response = await fetch('https://api.falconalert.com/users', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
  }
});

const profile = await response.json();
```

```python Python
import requests

url = 'https://api.falconalert.com/users'
headers = {
    'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
}

response = requests.get(url, headers=headers)
profile = response.json()
```

</RequestExample>

## Error Responses

| Status Code | Description |
|-------------|-------------|
| 401 | Unauthorized - Invalid or missing JWT token |

## Notes

- The password field is returned as a hashed value for security purposes
- The salt is used internally for password hashing and should not be displayed to end users
- Profile information is specific to the authenticated user making the request
