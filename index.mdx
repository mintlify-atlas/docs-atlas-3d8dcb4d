<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with FalconAlert API in minutes</p>
  
  <Steps>
    <Step title="Set up your environment">
      Clone the repository and install dependencies:
      
      ```bash
      git clone https://github.com/4rt21/backend-proyecto.git
      cd backend-proyecto
      npm install
      ```
    </Step>
    
    <Step title="Configure your database">
      Create a `.env` file with your MySQL credentials:
      
      ```bash
      MYSQL_HOST=localhost
      MYSQL_PORT=3306
      MYSQL_USER=your_user
      MYSQL_PASSWORD=your_password
      MYSQL_DB=Ofraud
      ```
      
      Import the database schema:
      
      ```bash
      mysql -u your_user -p Ofraud < backup.sql
      ```
    </Step>
    
    <Step title="Start the server">
      Run the development server:
      
      ```bash
      npm run start:dev
      ```
      
      The API will be available at `http://localhost:3000` and the Swagger documentation at `http://localhost:3000/docs`.
    </Step>
    
    <Step title="Make your first API call">
      Register a new user:
      
      ```bash
      curl -X POST http://localhost:3000/users/register \
        -H "Content-Type: application/json" \
        -d '{
          "name": "John Doe",
          "email": "john@example.com",
          "password": "securePassword123"
        }'
      ```
      
      <Accordion title="View response">
      ```json
      {
        "id": 1,
        "name": "John Doe",
        "email": "john@example.com",
        "username": "happy_john123",
        "created_at": "2024-03-04T12:00:00.000Z",
        "role_id": 1
      }
      ```
      </Accordion>
    </Step>
  </Steps>
</div>
