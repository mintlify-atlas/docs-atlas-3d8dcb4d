---
title: 'List Categories'
api: 'GET /categories'
---

Retrieves a list of all available categories in the system. Categories are used to classify reports.

## Authentication

No authentication required for this endpoint.

<CodeGroup>
```bash curl
curl -X GET https://api.falconalert.com/categories
```
</CodeGroup>

## Response

<ResponseField name="categories" type="array">
  Array of category objects
  
  <Expandable title="Category Object">
    <ResponseField name="id" type="number">
      Unique identifier for the category
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Name of the category
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Detailed description of the category
    </ResponseField>
  </Expandable>
</ResponseField>
